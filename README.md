# Bank Nifty Trading System

A simple and efficient automated trading system for Bank Nifty index.

## Overview

This project provides a cloud-based trading system for Bank Nifty, deployed on DigitalOcean for reliability and performance.

## Features

- Automated trading based on predefined strategies
- Real-time market data analysis
- Secure cloud deployment
- Easy configuration and management

## Deployment

For detailed deployment instructions, see [DEPLOYMENT.md](DEPLOYMENT.md).

Quick start:

```bash
export DIGITALOCEAN_TOKEN="your-token-here"
cd infra
terraform init
terraform apply
```
